# TheAuto Preview

TheAuto is a Windows automation project built to demonstrate visible desktop automation for a future stock-market analysis and paper-trading workflow. This preview repository only shows the public demo and project overview. The actual source code and private automation logic are not included here.

## Demo

🎥 Automation Preview: The system opens the Groww app automatically from Windows search and prepares the desktop workflow for the next automation steps.

## What This Project Shows

The current preview demonstrates the first working step of TheAuto: launching Groww through visible Windows automation. The goal is to build the system step by step so every action can be seen on screen, checked manually, and improved safely before moving toward advanced trading support.

## Project Vision

TheAuto is planned as a stock-market automation assistant for Indian equities. Around market open, it will observe sectors and stocks, identify strong movement, maintain a focused watchlist, and run paper trades using defined risk and cost calculations.

The project is designed to begin with paper trading only. Real-money trading will be considered only after consistent testing, reports, and manual review over multiple market days.

## Planned Workflow

- Open Groww automatically on Windows
- Allow manual login by the user
- Observe market movement during the early session
- Analyze strong sectors and stocks
- Update a focused watchlist
- Run paper-trading decisions
- Track buy, sell, hold, and switch decisions
- Generate daily performance reports
- Review strategy quality before any real-money use

## Safety Note

This preview is not financial advice and does not guarantee profit. The system is being developed for automation learning, market analysis, and paper-trading validation first. Any future real-money usage must include manual confirmation, risk limits, and proper testing.
